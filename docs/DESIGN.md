# TravelTalk 设计文档 v1.4

## 一、项目概述

TravelTalk 是一个移动端实时翻译 PWA，支持语音翻译、图片翻译和常用语三大功能。设计尺寸为 390×844px（标准移动设备屏幕）。

**当前版本**：v1.4（2026-04-24）—— 双 pill 回归 + 相机覆盖层 + 底卡只出现在相机模式。v1.3 的单 pill + 自动检测已全部回滚，但 TopBar / Settings / History 保留。

---

## 零、v1.4 布局变更速览

### 三页统一布局（65vh 主区 + 双 pill + 常用语）
```
Home（相机）             VoiceMode              
┌──────────────┐        ┌──────────────┐
│ TopBar       │        │ TopBar       │
├──────────────┤        ├──────────────┤
│              │        │              │
│ Viewfinder / │ 65vh   │ Chat Area    │ 65vh
│ PhotoOverlay │        │  (+ 右下 ➕)  │
│              │        │              │
├──────────────┤        ├──────────────┤
│[日本語][中文] │         │[日本語][中文] │
├──────────────┤        ├──────────────┤
│ 常用语横滑    │         │ 常用语横滑    │
└───(可选底卡)─┘         └──────────────┘
  ↑ 相机模式下 pill 录音/phrase → TranslationSheet 从底部滑入
```

### 双 pill 规格
- 布局：`flex items-center gap-3 px-4`
- 左 pill：`bg-[#1A1A1A] text-white`（点按 = 说外语 pair.B → 译到 pair.A）
- 右 pill：`bg-[#D8D2CA] text-[#1A1A1A]`（点按 = 说母语 pair.A → 译到 pair.B）
- 录音中：该 pill 变 `bg-[#D94F00] text-white` 显示 `● 点击结束`；另一 pill 禁用
- pill 显示：**源语言名**（`日本語`/`中文`），**无 icon**

### PhotoOverlay 规格
- 照片 `object-contain` 填满 65vh 容器
- 每个 OCR region：`position: absolute` + 百分比（原图坐标 → 容器百分比经 `locationToPercent` 映射）
- 气泡样式：`rgba(255,255,255,0.92)` 背景 + `border-radius: 6px` + `padding: 2px 6px` + `fontSize: clamp(10px, 2.6vw, 16px)`
- 左上角 toggle 按钮（🌐/👁）：切原图/译文叠加
- 右下角删除按钮（🗑）：清空照片回黑框
- OCR 加载中：半透明黑遮罩 + "识别中…" 白胶囊

### TranslationSheet 底卡
- 仅相机模式（`displayMode === 'photo'`）下，`isTranslating || translationError || (originalText && translationType in {'voice','phrase'})` 时渲染
- 方向来源：`isLeftRecording/isRightRecording` 实时；录音结束用 `lastSourceLang/lastTargetLang`

## 四、组件规范

### 4.1 语言切换胶囊 (voiceBar)

```
[中文 🎙]     [日语 🎙]
```

- 尺寸：宽度 fill_container，高度 52px
- 圆角：9999px（胶囊形状）
- 选中背景：`#000000`，文字白色
- 未选中背景：`#1C1C1E`，文字白色
- 间距：gap 12px

### 4.2 取景框 (viewfinder)

- 背景：`#000000`
- 圆角：16px（Home）或 20px（Recording）
- 图标：📷 emoji 60px 居中
- 提示文字：「拍摄菜单、路牌或标识」灰色 14px

### 4.3 模式切换器 (modeSwitcher)

```
[📷 拍照]  [🎙 语音]
```

- 外框：`#E0D9D0` 圆角9999px
- 内边距：2px
- 选中tab：`#000000` 背景
- 未选中tab：透明背景，灰色文字
- 切换动画：滑动指示器

### 4.4 常用语气泡

```
[多少钱] [地铁站] [可以刷卡吗]
[这个是] [谢谢] [洗手间在哪里] [+]
```

- 气泡背景：`#C5BEB6`
- 文字颜色：`#2D2D2D`
- 字号：13px
- 圆角：9999px（胶囊）
- 内边距：[0, 14]（水平）
- 高度：36px
- 间距：gap 8px
- + 按钮：透明背景 + `#C9C4BE` 描边

### 4.5 翻译底卡 (translationSheet)

- 背景：`#FFFFFF`
- 圆角：[20, 20, 0, 0]（底部圆角）
- 拖动把手：36×4px 圆角 `#C9C4BE`
- 原文：「你好，请问地铁站在哪里？」灰色 15px
- 译文：「すみません、地下鉄はどこですか？」深色 22px 粗体
- 播放按钮：180×48px 圆角 `#A8B5A0`
- **移动端滑动关闭**：向下滑动把手超过 80px 关闭底卡（已修复 iOS Safari 兼容性问题）

### 4.6 对话气泡（v1.2：加 ▶ 播放键）

**我的消息**（右对齐，深色背景）：
```
┌─────────────────────────┐ ▶
│ すみません、地下鉄は... │
│ 原文：你好，请问地铁站...│
└─────────────────────────┘
```
- 背景：`#1A1A1A`，文字白色 15px，圆角 `[4, 20, 20, 20]`
- 原文：灰色 10px
- ▶ 播放键：气泡右外侧 24×24px lucide `play` 图标，`fill: #888`
- 归属判定：`message.sourceLang === appStore.sourceLang`

**对方消息**（左对齐，浅色背景）：
```
▶ ┌─────────────────────────┐
  │ あちらの出口を右に...   │
  │ 原文：那个出口出来右转...│
  └─────────────────────────┘
```
- 背景：`#F2EDE8`，文字深色 15px，圆角 `[20, 4, 20, 20]`
- ▶ 播放键：气泡左外侧

**播放行为**：
- 点击 ▶ → `speakText(message.translatedText, message.targetLang)`
- **不用**全局 `appStore.targetLang`；避免切换当前语言后老气泡播错语种

### 4.7 "⟲ 新对话" 按钮（VoiceMode 专属）

- 位置：voiceBar 上方或右上角；24×24px lucide `rotate-ccw`
- 点击 → `setCurrentSessionId(null)` → 下一条翻译时自动创建新 session
- Home 模式**不**出现（Home 靠 15min idle 自动归档）
