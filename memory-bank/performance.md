# Performance Notes

## 当前关注点

- 语音链路延迟：PCM chunk 大小、发送节奏、尾部静音、`finish-task` 时序都会影响末句丢失和最终响应延迟。
- 图片翻译耗时：OCR + 批量 LLM 翻译应尽量一次调用完成，失败时提供 graceful fallback。
- 常用语命中速度：按目标语言 lazy 缓存，避免重复调用 LLM/TTS。
- 前端渲染稳定性：PhotoOverlay 的 bbox 百分比映射应避免因图片尺寸变化造成覆盖位置抖动。

## Agent 评估指标

- 任务完成率。
- 首次通过率 pass@1。
- 三次内通过率 pass@3。
- 每任务重试次数。
- 每任务耗时。
- 失败原因分类：需求理解、环境、测试、集成、外部服务。

## 后续可量化项

- 语音录音结束到 `complete` 帧的 P50/P95。
- 图片上传到覆盖层渲染的 P50/P95。
- 常用语缓存命中率。
- 前端首屏加载与 PWA 离线可用性。
